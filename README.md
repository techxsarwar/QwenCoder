# ⚙️ Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/QwenCoder.git
cd QwenCoder
```

---

## 2️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 3️⃣ Set Environment Variables

Create `.env`

```env
TELEGRAM_BOT_TOKEN=YOUR_TOKEN
QWEN_API_KEY=YOUR_API_KEY
```

---

# ▶️ Run Bot

```bash
python bot.py
```

---

# ☁️ Deploy on Render

## Build Command

```bash
pip install -r requirements.txt
```
